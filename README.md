# 🍽️ Food Mood

A recipe discovery web application that helps users find delicious meals based on ingredients or categories, with a clean and interactive user interface.

---

## ✨ Features

- 🔍 Search meals by ingredient (e.g., chicken, rice, egg)
- 🏷 Filter meals by category
- 🎲 Random meal generator
- 📋 View detailed recipe information
- 🥗 Ingredients & measurements list
- 🧑‍🍳 Step-by-step cooking instructions
- ▶️ YouTube video tutorial (if available)
- ❌ Clear results functionality
- ⚠️ Error handling for invalid inputs

---

## 🧠 Project Idea

Food Mood was designed to simplify the process of finding recipes by allowing users to explore meals based on what they already have or what they’re interested in.

The goal was to create a user-friendly experience that combines:
- Simplicity
- Interactivity
- Useful real-world functionality

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- API Integration (Meal API)
